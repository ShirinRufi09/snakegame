Snake Game
A desktop-based Snake Game built using core Java, demonstrating strong fundamentals in object-oriented programming, multithreading, event handling, and GUI development. The project is structured with modular classes to manage game logic, keyboard input, rendering, and thread control, resulting in smooth and responsive gameplay.

Features
Real-time snake movement using keyboard controls
Dynamic food generation and score tracking
Collision detection with walls and snake body
Multithreaded game loop for consistent speed and responsiveness
Clean, modular object-oriented design
